This is a test of the gh-pages branch of a github repository

Using the "gh-pages" branch

The purpose of this repo is to just simply test out first hand how the "gh-pages" branch is handled on github. By Placing a "gh-pages" branch in a repo of mine and pushing the branch to the remote here on github, the name of the repo will apear as a subfolder on my github page.

In other words the absolute path of the "gh-pages" branch of this repository called "gh-pages-test", under my github user name "dustinpfister" should be at https://dustinpfister.github.io/gh-pages-test/index.html

Using cdn.rawgit.com

I can also use rawgit to link to an web project in a repo.

Linking to any branch

Raw git link to the "master" branch

https://cdn.rawgit.com/dustinpfister/gh-pages-test/master/index.html

Raw git link to the "gh-pages" branch

https://cdn.rawgit.com/dustinpfister/gh-pages-test/gh-pages/index.html

Linking to a certain commit.

Raw git link to the my first commit in the master branch where "5b1d5c3df6c8c60f2a17ed834b2ee5fb9cf3836b" is the id of that commit.

https://cdn.rawgit.com/dustinpfister/gh-pages-test/5b1d5c3df6c8c60f2a17ed834b2ee5fb9cf3836b/index.html

Using raw.githubusercontent.com

If for some reason I want to link to content in a repo that is given as plain text, rather than html. I can use raw.githubusercontent.com. https://raw.githubusercontent.com/dustinpfister/gh-pages-test/master/index.html
